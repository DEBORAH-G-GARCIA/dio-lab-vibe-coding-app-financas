# App de Finanças Pessoais da Deborah 

Este projeto tem como objetivo criar um aplicativo de organização de finanças pessoais que funciona por meio de conversas em linguagem natural.  
A proposta é oferecer uma experiência simples, acessível e prática para quem deseja começar a controlar seus gastos sem complicações.

---

## PRD Refinado no Copilot Web

### Contexto
Criar um aplicativo de Organização de Finanças Pessoais que funcione por meio de conversas em linguagem natural.  
Objetivo: facilitar o controle financeiro sem formulários manuais ou planilhas complexas.

### Problema
Usuários desistem de controlar seus gastos porque os apps atuais exigem muita entrada manual e oferecem pouca personalização.  
Solução proposta: experiência de conversa com recomendações automáticas de economia.

### Público-Alvo
Pessoas que desejam começar a organizar suas finanças de forma prática e sem complicação, especialmente iniciantes.

### Funcionalidades-Chave
1. Registrar gastos via chat em linguagem natural.  
2. Classificar automaticamente as transações.  
3. Definir e acompanhar metas financeiras.  
4. Receber dicas de economia do “Agente Financeiro”.  
5. Visualizar relatórios simples e personalizados.

### Entregável da IA
Gerar um plano de MVP com:
- Principais telas (chat, metas, relatórios).  
- Recursos necessários (NLP para interpretação, categorização automática, motor de recomendações).  
- Esboço de validação inicial (testes com usuários iniciantes, feedback sobre clareza e utilidade).  

Tom: educativo e linguagem acessível em português.

## Resumo das Funcionalidades

### Registro de Gastos via Chat
- Usuário informa seus gastos em linguagem natural.
- Exemplo: "Gastei 50 reais com comida".
- O assistente interpreta e registra automaticamente.

### Classificação Automática
- Transações são categorizadas em áreas como "Alimentação", "Outros", etc.
- Dispensa formulários e planilhas manuais.

### Saldo Atual
- Exibição em tempo real do saldo do usuário.
- Inclui situações de saldo negativo.

### Metas Financeiras
- Aba dedicada para definir objetivos de economia.
- Acompanhamento do progresso das metas.

### Relatórios Simples e Personalizados
- Aba de relatórios para visualizar gastos por categoria.
- Permite acompanhar tendências de forma clara e acessível.

### Interface Intuitiva
- Navegação por abas: **Metas**, **Chat** e **Relatórios**.
- Ícones de acesso rápido para facilitar a experiência.
